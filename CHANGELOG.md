## 版本说明

### 版本号格式

`主版本号。次版本号。修订号`

- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向后兼容的功能新增
- **修订号**: 向后兼容的问题修正

### 变更类型说明

- **Added** - 新增功能
- **Changed** - 变更现有功能
- **Deprecated** - 即将移除的功能
- **Fixed** - Bug 修复
- **Security** - 安全更新
